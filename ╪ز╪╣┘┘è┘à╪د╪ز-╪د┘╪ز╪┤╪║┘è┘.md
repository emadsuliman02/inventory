# نظام جرد الأصول التقنية — سيرفر محلي على شبكة الجمعية

هذا النظام يشتغل بالكامل على جهازك (بدون إنترنت، بدون Claude، بدون قوقل). أي موظف على نفس شبكة الجمعية يقدر يفتحه من متصفحه العادي.

---

## 1) قبل التشغيل: عدّل كلمتي المرور

أول تشغيل للسيرفر على أي جهاز جديد ينشئ تلقائيًا ملف `accounts.json` بجانب `app.py` فيه كلمتا مرور مؤقتة. افتحه بأي محرر نصوص (Notepad مثلًا) وعدّله لهذا الشكل:

```json
{
  "administrator": { "password": "كلمة-مرور-مسؤول-النظام", "role": "admin" },
  "admin": { "password": "كلمة-مرور-المستخدم-العادي", "role": "user" }
}
```

- حساب `administrator` (دور مسؤول النظام، يشوف القسم الآمن)
- حساب `admin` (دور مستخدم عادي)

احفظ الملف، ثم أعد تشغيل `python app.py` عشان التغيير يسري. ملاحظة: `accounts.json` ملف محلي على هذا الجهاز فقط ولا يُرفع لأي مستودع (Git) — كل جهاز يشغّل السيرفر له نسخته الخاصة.

---

## 2) ثبّت البرامج المطلوبة (مرة وحدة بس)

1. ثبّت Python من الموقع الرسمي: https://www.python.org/downloads/
   - أثناء التثبيت على ويندوز، **تأكد تفعّل خيار "Add Python to PATH"** أسفل نافذة التثبيت.
2. افتح "موجه الأوامر" (Command Prompt / Terminal)، وروح لمجلد المشروع:
   ```
   cd المسار\الى\مجلد\it-assets-server
   ```
3. ثبّت المكتبة المطلوبة:
   ```
   pip install -r requirements.txt
   ```

---

## 3) شغّل السيرفر

بنفس نافذة الأوامر:
```
python app.py
```

بيطلع لك رسالة تأكيد إن السيرفر شغال، واتركه شغال (لا تسكر النافذة).

---

## 4) اعرف عنوان IP لجهازك على الشبكة

- **ويندوز:** افتح موجه الأوامر واكتب `ipconfig`، ودوّر على "IPv4 Address" (مثال: `192.168.1.10`)
- **ماك/لينكس:** افتح Terminal واكتب `ifconfig` أو `ip addr`، ودوّر على عنوان يبدأ بـ `192.168.` أو `10.`

---

## 5) شارك الرابط

أعطِ أي موظف على نفس شبكة الجمعية هذا الرابط (استبدل الرقم بعنوان IP جهازك من الخطوة 4):
```
http://192.168.1.10:5000
```

يفتحه من متصفح جواله أو كمبيوتره العادي، ويسجل دخول بالحساب المناسب.

---

## ملاحظات مهمة

- **الميزات الجديدة:** أضفنا تبويبات "السيرفرات"، "أجهزة الأمان"، و"الأبواب الذكية" — كل واحد فيه: الاسم، الموقع، الدور، بعهدة من (موظف)، الموديل، المواصفات، والحالة (متاح / قيد الاستخدام / تحت الصيانة). كل جهاز فيه زر 📜 يعرض "سجل العهدة" — قائمة بكل من حازه سابقًا مع تاريخ التسليم.
- **خلّي الجهاز شغال ومتصل بالشبكة** طول الوقت اللي تبي فيه النظام متاح للجميع. لو سكرت البرنامج أو الجهاز، الموقع يوقف عن العمل للجميع لحد ما تشغله ثانية.
- **البيانات تُخزّن في مجلد `data/`** (ملفات JSON) بجانب `app.py`. خذ نسخة احتياطية من هذا المجلد بين فترة وأخرى (انسخه لفلاش أو مكان ثاني).
- هذا السيرفر مصمم للاستخدام **داخل شبكة الجمعية فقط** (لا يظهر على الإنترنت العام)، وهذا مقصود لحماية البيانات الحساسة.
- لو حبيت النظام يشتغل تلقائيًا كل مرة يشتغل فيها الجهاز (بدون ما تفتح موجه الأوامر يدويًا)، هذا ممكن لاحقًا عن طريق إعداد "خدمة ويندوز" (Windows Service) — قول لي إذا تبي تفاصيلها.
